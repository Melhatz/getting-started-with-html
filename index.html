HTML

HTML stands for Hypertext Markup Language and it is the most common type of file found on the web. HTML documents are made of:

  • Text Content- What you see on the page
  • Mark Up Or Styling- How the content looks on the page
  • Refrences to other documents on the web- examples: Video, Audio or Image files
  • Links to other html pages on the web

    Inline vs Block Elements

    Elements can be broken down in to 2 types:
    Inline: Elements that flow in text content and will follow white space settings.
    OR
    Block: Elements that create invisable blocks around content that can have special padding and margins.


    HTML Documents

    HTML documents have a head and body area. The head contains the title and styling information. The body contains the page content that is to be displayed to the user.
